# WSU Design Standards Knowledge Base

**For Claude agents:** Read this index first. Use the CSI code to locate files, then read individual files for full detail. All paths are relative to `standards/`.

**Source:** WSU Facilities Services Design Standards, June 2025 ZIP archives. CSI Divisions 02-40, campus-wide guidelines, supplemental guides, and detail drawings.

## How to Use

1. Find the CSI division or topic below; note the file path.
2. Read the full standard (e.g., `standards/wsu/division-26/26-00-00-electrical.md`).
3. Multi-part files cover one document split for size; read all parts for completeness.
4. Archived files are superseded -- use current versions unless researching history.
